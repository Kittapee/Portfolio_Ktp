ASMS Project

Aluminium Stock Management System

Project Overview

โปรเจกต์ระบบจัดการคลังสินค้าประเภทอะลูมิเนียม (Aluminium Stock Management System - ASMS) เป็นโปรเจกต์ระยะยาวที่พวกผมพัฒนาขึ้นเพื่อแก้ปัญหาการจัดการสต็อก ผู้จัดทำได้มีกันทั้งหมด 3 คน โดยที่มีผมเป็นหนึ่งในสมาชิก โดยแบ่งการพัฒนาออกเป็น 2 เฟสหลักๆ ตลอดระยะเวลา 3 ปีที่ผ่านมาในมหาวิทยาลัยเทคโนโลยีราชมงคลอีสาน และในอนาคตในช่วงของการเรียนปีที่ 4 จะเป็นการ Deploy ผลงานจริงสู่ธุรกิจ

ความรับผิดชอบ

เราได้มีการพัฒนาร่วมกันในเฟสต่างๆ ของโปรเจกต์นี้ โดยที่ผมจะเน้นไปที่การสร้างเว็บแอปพลิเคชันสำหรับจัดการสต็อกอะลูมิเนียม

Phase 1: ปี 3 เทอม 1

Database Management Systems

เริ่มต้นจากการศึกษาเกี่ยวกับการวิเคราะห์ความต้องการของผู้ใช้งาน ระบบที่มีอยู่และการกำหนดขอบเขตของระบบใหม่ โดยที่จะเน้น การวางรากฐานระบบฐานข้อมูลในวิชา Database Management Systems(DBMS) ในปีที่ 2 โดยการออกแบบตาราง ความสัมพันธ์ และ Data Flow เพื่อให้ระบบเก็บข้อมูลเข้า-ออกได้อย่างแม่นยำและไม่ซ้ำซ้อน

Phase 2: ปี 3 เทอม 2(ปัจจุบัน)

System Analysis

ต่อมาในปีที่ 3 สำหรับการวิเคราะห์และออกแบบระบบ System Analysis(SA) คือการทำต่อจากวิชา DBMS โดยจะเป็นการนำข้อมูลที่ได้จากการวิเคราะห์ความต้องการของผู้ใช้งานและระบบที่มีอยู่ในเฟสแรก มาวางโครงสร้างและออกแบบระบบใหม่อย่างละเอียด โดยในเนื้อหาที่เรียนจะมีการศึกษาเกี่ยวกับการวิเคราะห์ความต้องการของผู้ใช้งาน ระบบที่มีอยู่ และการกำหนดขอบเขตของระบบใหม่อย่างละเอียด
